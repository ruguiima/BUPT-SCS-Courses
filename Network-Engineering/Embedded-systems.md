# 嵌入式系统（必修）

## 评价1

1. **开课信息**：前8周理论课：戴志涛，后8周实验课：刘健培（2025秋）
2. **授课内容**：
   - 前8周是老熟人戴志涛孜孜不倦讲他那个深蓝色ppt，关于嵌入式系统理论的一些内容。总之0人在意，和实验课半毛钱关系没有，甚至和实验用的板子型号对不上。
   - 后8周是刘老师下发实验用开发板，一周配环境，四周做4个小实验，最后3周1-4人组队做一个大实验。
3. **考核方式**：
   - 没有期末考试。理论课分数占比疑似无限趋近0，没有签到没有课后作业，有两次当堂作业但是我全翘了貌似也没有什么影响。
   - 实验课分数构成：4 * 平时实验（60%）+ 大实验（40%）  
4. **作业情况**：
   - 在鸿蒙开发板`Hi3861`上完成4次个人小实验和一次组队大实验。小实验需提交实验报告，源代码和演示视频，大实验当堂验收，只需提交实验报告和讲解视频。使用c语言编程，IDE用`vscode`。
   - 第一次实验课老师会提供实验环境和安装说明，demo代码资料和讲解文档以及辅助软件等。小实验可学习资料中的demo代码并仿照其编写，复用其中部分现有模块并充分利用ai即可，难度不高。
   - 大实验为四个基础功能（20%）+四个附加功能（20%）。四个基础功能实现较简单，加上附加要求稍有难度吧。我们3个人做了整三天左右完成了全部功能，调试有点痛苦，总体工作量适中吧。
5. **成绩情况**：95
6. **心得体会**：
   - 实验均可自己在课余时间完成，因此这门课理论上整个学期只需要去两次：第一次实验课领开发板，做好大实验后挑一周去验收并归还板子。
   - 前8周理论课的存在真是个迷，实际体感就是前半学期这门课和不存在一样，后半学期却又内容太多忙不过来。
   - 小实验均提交，大实验完成所有的附加要求应该就可以在本课程获得较高分数。总体一门很干脆爽快的课程。
