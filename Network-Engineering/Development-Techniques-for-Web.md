# Web开发技术（必修）

## 评价1

1. **开课信息**：王尊亮老师（2019秋）
2. **推荐等级**：强烈推荐
3. **授课内容**：web开发概述，HTML&CSS&JS介绍与应用，Servlet，SpringMVC，Spring Data等技术，用于入门Java Spring框架
4. **考核方式**：期末开卷考试，平时三四次代码小作业，和三人组队大作业（写一个知乎Demo）
5. **作业情况**：一共三四次小作业，是围绕构建通讯录为主题逐渐丰富，所以只需要增量开发代码即可，写一个知乎Demo大作业相对复杂
6. **成绩情况**：期末占比50%，大作业30%，小作业20%，期末考试题虽然开卷，但内容较多，甚至可能没有接触过，认真学80+肯定有，给分中庸
7. **心得体会**：要真正去拥抱一门技术，并实践，通过这门课学到了很多东西，也为大三下的选修课web后端开发奠定了基础
8. **其他方面**：王老师为人非常好，给学生非常nice的指导与答疑，个人的web开发技术也是非常NB，同时对业界的开发模式非常了解，我想真正的好老师大抵如此吧

## 评价2

1. **开课信息**：王尊亮老师（2025秋）
2. **推荐等级**：比较推荐
3. **授课内容**：web前端（`HTML&CSS&JS`）介绍，`SpringMVC`，`thymeleaf`，`MyBatis`等后端或前后端交互技术讲解，用于入门Java Spring框架。老师上课还会示范使用IDE的ai插件进行开发，并介绍一些新颖的ai开发插件和框架。
4. **考核方式**：平时（考勤、平时作业、综合实践作业）50% + 期末开卷考试 50%
5. **作业情况**：
   - 平均每两章一次使用飞书文档编写的作业，分为问答题和实验编码题，一学期总共5次。想省事可以快速用ai做完，认真写工作量比较大，但是确实能掌握基本的web开发知识，基本涵盖了上课的核心内容。
   - 临近期末会布置一次大作业，个人自主选题，使用ai协同开发一个小型Web系统。工作量不算大，主要目的是学习自主开发的基本流程。
6. **成绩情况**：85+
7. **心得体会**：
   - 本课程给分中庸，期末开卷考试以概念考察为主。个人感觉对成绩的影响——web技术栈掌握程度 < 多准备点开卷资料。可打印的资料有上课ppt，飞书作业文档，往年题等，基本涵盖了期末考题的内容。
   - 总体而言，是一门划水很轻松，认真学也能学到真才实学，但是对保研党刷分性价比较低的课程。
